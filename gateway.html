SIMATIC FDE Gateway

Features

  • Multiple device interfaces onboard

  • Ethernet based MQTT as IT communication

  • No PLC or OT interference

  • Temperature range from –30°C to 60°C

  • One cabinet DU (Division Unit)

  • Configuration and data selection via webserver

  • Secure certificate based communication

Value

  • Multiple device interfaces onboard

  • Data selection via webserver w/o any further tooling

  • Very compact and robust

  • Secure communication towards IT by design

  • Do not interfere PLC or OT system (especially for brownfield applications)

  • Gateway can be placed directly next to IO Systems (directly in the field)

  • Standard IT communication for field data to publish information on any data sink

  • Get data and insights from non field bus connected devices
